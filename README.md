## 구현 내용

+ DatePicker를 이용하여 타이머를 설정할 수 있다.

+ 시작버튼을 누르면 타이머가 시작된다

+ 일시정지 버튼을 누르면 타이머가 일시정지 된다.

+ 취소 버튼을 누르면 타이머가 종료된다.

+ 설정한 시간이 다 지나면 종료되고 알람이 울린다.


## 학습 내용

+ DispatchSourceTimer 기능을 이용하여 타이머를 구현
